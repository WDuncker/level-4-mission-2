## About The Project

This mission was my first foray into using Javascript on a website using basic HTML and CSS. I chose to build on my datasheet that I created in Mission 1 describing the details for a Gundam from an anime I love.

## Roadmap

- [ ] Built on the existing page by separating the different facets of the mobile weapon into home, overview, armaments and systems & features pages.
- [ ] Added javascript to create an automatic carousel showcasing the suit in action and creating clickable dropdown menus on the subpages to make it look tidy and only show information that the user is interested in.

See the [open issues](https://github.com/WDuncker/level-4-mission-2/issues) for a full list of proposed features (and known issues).
